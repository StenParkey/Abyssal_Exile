# 🦴 Welcome to the Repo for Abyssal Exile 🦴
As per the repo description, this is a Terminal Based, **C++**, Choose your own Adventure Horror Game. 

☠️ The adventure takes place in a horrible void, scathed in darkness and home to terrible entities. It is unclear how you ended up here, or if escape will ever be possible. Experience the realm if you dare... ☠️

I made it for my submission to the Text Adventure Project for Codecademy's **Learn C++ Course**. It was a lot of fun, and I was happy with my results. Clone the repository and try it out for your self!

There are a few issues with it currently. The main one being.. `If you input a string to one of the question inputs. It will break the running state loop and spam the terminal with the invalid input message`
When encountering this issue, just use control + break (ctrl+c on Windows and Linux, and cmd+c on Mac) to stop the program 
I'll fix the issue as well as implement reusable functions later as I learn more c++.

##### Clone it with:
```bash
git clone https://github.com/StenParkey/Abyssal_Exile
```
Then run the executable in your terminal and enjoy the void... 🕯️🕯️🕯️
